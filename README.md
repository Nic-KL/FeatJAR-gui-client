# FeatJAR-Gui-Client based on Eclipse GLSP - Client
This project is a *standalone client* which can be used in combination with the [FeatJAr-gui](https://github.com/Nic-KL/FeatJAR-gui-new) project to display feature models.


## Gradle

### First run 
```bash
./gradlew yarnInstall
```

### Always (if the TypeScript code has been modified)
```bash
./gradlew yarnBuild
```

### You can also run both commands via
```bash
./gradlew build
```


## Traditional using Yarn

### First run
```bash
yarn install
```

### Always (if the TypeScript code has been modified)
```bash
yarn build
```
